# Análise de Dados com Python

Este projeto consiste na análise e manipulação de dados utilizando Python e Jupyter Notebook. O objetivo é aplicar conceitos de leitura de arquivos CSV, tratamento de dados e geração de análises a partir de um conjunto de informações.

## Tecnologias Utilizadas

- Python 3  
- Jupyter Notebook  
- Pandas  
- CSV  

## Estrutura do Projeto

- Letterboxd_Manager.ipynb
- exemplo.ipynb
- letterboxd_movies_dataset.csv

## Funcionalidades

- Leitura de arquivo CSV  
- Tratamento e organização dos dados  
- Criação de novas colunas e métricas  
- Filtros por idioma, ano e duração  
- Análises quantitativas
